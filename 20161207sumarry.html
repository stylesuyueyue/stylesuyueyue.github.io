ECMAScript编程规范

命名规范：

1.所有变量必须是有意义的英文，严厉禁止拼音。

2.变量命名采用小驼峰法（第一个单词首字母小写，其余单词首字母大写）

3.大括号的位置：起首的大括号跟在关键字的后面:block {
                                        　...
                    　}

4.遵守同一套编码命名规范

5.最后写出的编码风格要像一个人写出来的

function函数

函数声明的三种方法和区别： 
1.函数声明:function foo(){}

该函数是javascript里的一等公民，不论在哪里调用都会被提到顶端。

2.表达式声明：var foo=function(){}
 分析：   
            	   foo()      //输出undefined//表示已声明，但未赋值
                var foo=function(){
                	alert(111);
                }
                 foo();  //正确调用              
            
3.匿名函数：function (){}
                           该函数调用时需去掉严谨模式。 
               (function (){
               	  alert(222);
               })();        //该函数调用方法
                            还可以这样((function (){
               	  alert(222);
               })());     //不需要掌握

注意事项：

1.函数声明：若有两个相同函数被调用，则后一个函数覆盖前一个函数。

2.表达式声明优先权高于函数声明，即当相同函数以表达式声明和函数声明同时存在时，表达式声明覆盖函数声明，不论位置是否在后。

3.var foo=function fun(){
            	  console.log(fun);   //正确自调用
            }
                // fun();   //调用无效,报错
                console.log(fun);    //报错
                foo();    //正确调用，显示整个函数
            
4.var str="(function(){alert(11);})()"
                  eval(str);   //将字符串解析为表达式或值function (){ alert(11); }，之后运行
