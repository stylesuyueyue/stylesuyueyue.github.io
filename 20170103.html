(1)点击切换内容：

	a.鼠标移入时，红色条跟随移动，同时切换到相应照片。
	b.为所选目标增加addClass("active")，同时将其兄弟的active取消siblings(".tab-item").removeClass("active")。
	c.获取移入目标的索引$(".main").eq($(this).index())，用其找到对应的图片，为其增加addClass("selected")，同理也要取消其兄弟的selected：
	   siblings(".main").removeClass("selected")。
	d.eq(i):寻找到索引的值；siblings(".main"):寻找出本身的兄弟元素。
	e.主要代码：$(function(){
    		$(".tab li").mouseenter (function(){
    			$(this).addClass("active").siblings(".tab-item").removeClass("active");
    			var ind =$(this).index();
    			console.log(ind)        //索引值
    			$(".main").eq(ind).addClass("selected").siblings(".main").removeClass("selected")
    		})
 })点击切换内容实例查看

(2)轮播：

	a.鼠标移入数字是，轮播到相应图片。
   	b.为所选目标增加addClass("active")，同时将其兄弟的active取消siblings().removeClass("active")。
	c.通过索引值获得对应图片，为其加上CSS。
	d.主要代码：var idx =0;
			$(".wrap ol li").mouseenter(function(){
				idx +=1;
				$(this).addClass("active").siblings().removeClass("active");
				$(".wrap ul li").eq($(this).index()).css({
					"left":650,
					"z-index":idx,
				}).animate({
					"left":0,
				},300)          })     轮播实例查看

(3)手风琴：

	a.首先用each为其添加背景图片。
   	b.设置鼠标移入移出时，宽变化。
	c.设置时，注意要清除影响。
	d.主要代码：var idx =0;
		$(".wrap li").each(function (i,ele) {
                $(this).css("background-image","url(img/" + (i + 1) +".jpg)");
            }).mouseenter(function () {
                $(this).stop().animate({
                    "width":600
                },300).siblings().stop().animate({
                    "width":200
                },300)
            }).mouseleave(function () {
                $(this).stop().animate({
                    "width":200
                },300)     });     手风琴实例查看

		window.onscroll = function() {
			var tp = $(document).scrollTop();

			}, 300)

				$(".order").hide()
